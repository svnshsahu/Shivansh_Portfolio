# 💼 Shivansh Sahu – Developer Portfolio

A modern, responsive developer portfolio built with **React** and **Tailwind CSS** to showcase my projects, technical skills, education, and experience. The portfolio features a clean UI, smooth animations, and dedicated project pages with detailed information.

## 🚀 Live Demo

**Portfolio:** https://www.shivanshsahu.website/

---

## ✨ Features

* Modern and responsive design
* Smooth scrolling and clean navigation
* Dedicated About, Skills, Projects, and Contact sections
* Individual project detail pages
* GitHub and LinkedIn integration
* Mobile-friendly interface
* Glassmorphism-inspired UI
* Fast performance with Vite
